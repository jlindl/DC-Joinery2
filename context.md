## 6. Strategic gaps and opportunities

Useful to flag if this analysis feeds work for the business:

1. **No website.** A simple portfolio site would significantly improve discoverability for "bespoke joiner Newcastle" / "fitted furniture Jesmond" type searches and lend credibility for higher-value commissions.
2. **No Google Business Profile.** Free, high-impact for local "near me" searches and Maps visibility.
3. **No trade-directory presence.** Checkatrade / Houzz Pro accounts could compound the existing 100% recommendation rate into ranked visibility.
4. **Email is personal (`@me.com`).** A branded address (`hello@dcjoinery.co.uk` or similar) would raise perceived professionalism, especially for commercial enquiries.
5. **Domain check needed.** `dcjoineryandinteriors.co.uk` and `.com` availability worth verifying before any rebrand/website work.
6. **Companies House registration** would unlock VAT-registered commercial work and limit personal liability — relevant if turnover is approaching the VAT threshold (£90k as of 2024).
